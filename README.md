# Earthquakes & Tropical Cyclones Monitor

Real-time dashboard showing:
- **Earthquakes** (last 30 days, M≥1.0) – USGS
- **Tropical Cyclones** (active + recent) – NOAA NHC + JTWC

Live Demo: [Open App](https://your-username-your-repo-name.streamlit.app)

## Features
- Interactive world map
- Depth, tsunami, felt reports
- Storm category, wind speed, basin
- Clickable official links
- Auto-refresh every 30 min

## Data Sources
- [USGS Earthquake API](https://earthquake.usgs.gov)
- [NOAA NHC](https://www.nhc.noaa.gov)
- [JTWC](https://metoc.navy.mil/jtwc/)
